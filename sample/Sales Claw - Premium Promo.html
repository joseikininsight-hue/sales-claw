なぜ Sales Claw なのか

従来の営業自動化ツールと異なり、Sales ClawはAI CLIが主体で動く。フォームごとに構造を解析し、その会社にしか刺さらないメッセージを生成する。

企業サイト深掘り分析
company-analyzerがサイト全体を巡回し、事業領域・ギャップ・注力分野を自動抽出。テンプレートではなく、その企業のためだけの文面を生成します。
→ 分析 + メッセージ生成まで自動
Electron内フォーム自動入力
WebContentsViewでフォームを直接操作。セレクタマッピング → 入力 → スクリーンショットまで全自動。ユーザーはダッシュボードで確認して送信するだけ。
→ 入力後スクショ必須 · 送信は人間が判断
3AI プロバイダー対応
Claude Code / OpenAI Codex / Gemini CLIの3つをUI上で切り替え可能。OMCモデルルーティングでステップ別に最適なモデルを自動選択し、コストを60%削減。
→ haiku / sonnet / opus で自動コスト最適化
リアルタイム ライブモニター
CLI ActivityフィードとWebSocket SSEでリアルタイム進捗表示。各社の処理ステータスをチャット風UIで確認。フロートパネルで常時監視可能。
→ フォーム入力中の進捗もリアルタイム反映
並列フェーズ処理
フェーズA（分析）は全社を並列サブエージェントで同時処理。フェーズB（フォーム入力）は順次実行。2フェーズ設計で最大スループットを実現。
→ N社を並列分析 → 順次入力 → 完全自動化
承認ワークフロー
入力済みフォームのスクリーンショット・本文・安全チェックをダッシュボードで一覧表示。人間が内容を確認してから送信。CAPTCHA・営業NG企業は自動スキップ。
→ 人間の判断を最終ゲートとして確保
ダークモード対応

深夜の営業作業も、目に優しく。

ダークテーマは単なる外観ではなく、長時間作業での疲労軽減のための設計です。
